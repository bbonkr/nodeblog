## Usage

Copy `src/.sample.env` to `src/.env` then edit your `src/.env`.

Need to fill database information.

> ### .env Sample
>
> for MariaDB
>
> ```
> SITE_NAME=nodeblog
> COOKIE_SECRET=nodeblog
> DB_HOST=10.10.0.10
> DB_PORT=3306
> DB_DATABASE=nodeblog
> DB_USERNAME=nodeblog
> DB_PASSWORD=nodeblog
> SKIP_PREFLIGHT_CHECK=true
> ```

```sh
npm run start
```

## Run tests

```sh
npm run test
```
